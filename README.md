# LRU-Cache-Java
# LRU Cache Implementation (Java)
# Project Overview

The Least Recently Used (LRU) Cache is a data structure used to store a limited number of items and remove the least recently used item when the cache reaches its capacity.

This project implements an efficient LRU Cache using Java and appropriate data structures to achieve optimal performance.

# Objectives

Implement LRU cache logic

Manage cache memory efficiently

Improve data retrieval performance

# Technologies Used

Java

Data Structures

HashMap

Doubly Linked List

# Working Process

Store elements in the cache with a fixed capacity.

When a new element is added and the cache is full, the least recently used element is removed.

When an element is accessed, it becomes the most recently used item.
